1. Pre-requirements for AMPLIX

Install does not meet required environment to proceed.
Please check bellow steps for install and try again.

  1. Install the Java SDK(1.5.0 or later).
  2. Install Apache Tomcat (5.5 or later).
  3. Install MYSQL database
  4. Download the latest version mplix.war, mplix_app.zip from amplixbi.com, and unzip it.
  5. Setup and start web application as described in section 4.
  6. Use installation wizard http://localhost:8080/mplix/install.html, for guided install sample databases and demo.

2. Install AMPLIX analysis server.

  1. From the unzipped binary release, explode mplix.war to TOMCAT_HOME/webapps/mplix
  2. From the unzipped binary release, explode mplix_app.zip to IGC_HOME (Pre-packaged on C:\mplix)
  3. Open the web.xml file in TOMCAT_HOME/webapps/mplix/WEB-INF and edit IGC_HOME for the folder IGC_HOME from the instructions above.
  4. Open the config.xml file in IGC_HOME/config and edit database userid and password for your mysql instance.
  5. After install, you need to remove folder IGC_HOME/install.

3. Prepare database

  1. Start database, if needed
  2. Login as root user with the following command.

    mysql -uroot -ppassword

  3. Create empty database with the following command on the above database command shell.

    create database mplix;

4. Deploy and run the web appication

  1. Start up Apache Tomcat.
  2. Go to http://localhost:8080/mplix/install.html, setup wizard and follow instructions.
