Thank you for choosing AMPLIX.

Before install AMPLIX

  1. Open the config.xml file in IGC_HOME/config and edit database userid and password for your mysql instance.
  2. Start database, if not started yet
  3. Login as root user with the following command.

    mysql -uroot -ppassword

  4. Create empty database with the following command on the above database command shell.

    create database mplix;

  5. After install, you need to remove folder IGC_HOME/install.
