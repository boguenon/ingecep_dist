Server Configuration

Server Settings:

Setting up AMPLIX! to run on your server involves admin email settings here.

The first step is email configuration. Choose your preferred settings:

Email Settings

Email is used to notify password change as a given email address.

Single Sign On(SSO) Settings

SSO enables LDAP central login validation. The customized java class file for validating LDAP is necessary and load dynamically when Orbit starts.

Email Settings
This is SMTP email server that could send email from Orbit. Examples on Gmail is smtp.gmail.com

SMTP email server port number to connect with. Standard email SMTP server port number is 25 and Gmail is 587

This can be the default SMTP username, a username provided by your email provider.

Using a password for the SMTP account is mandatory for site security. This is the same password used to access your email.


Advanced Settings


Enable
Use SingleSignOn class to varify username and password.

Customized Java class file. This class is loaded dynamically when server starts up. When user try to login server, this class varify username and password.
